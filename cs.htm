界面

登录界面


- : 账号.

- : 密码.

- : 登录.

- : 注册帐号.

- : 修改密码.

- : 虚拟键盘我.

- : 退出游戏


服务器选择界面


- 鼠标点击或数字键盘(0~9)选择服务器.

- 0区为成人服务器,未到20岁不可连接.

- 每个服务器可以建立三个角色.

- 服务器上的角色是独立于各个服务器的.


人物选择界面


- : 以当前人物开始游戏.

- : 创建新人物.

- : 删除人物.

- : 返回服务器选择界面.

- : 退出游戏.


角色创建界面


- : 人物名称.

- : 人物职业(战士/法师/道士/刺客).

- : 人物性别(男/女).

- : 创建人物.

- : 退出角色创建界面.
